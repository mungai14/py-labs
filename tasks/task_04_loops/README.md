## Your Task

### Part A ‚Äî Countdown
Write a program that asks the user for a number, then counts **down** from that number to 1, printing each number. At the end print `"Blast off!"`.

### Example output
```
Enter a number: 5
5
4
3
2
1
Blast off!
```

### Part B ‚Äî Guessing Game
Write a guessing game where:
- The secret number is `7`
- The user gets **4 attempts**
- Each wrong guess prints `"Wrong, try again."`
- A correct guess prints `"You got it!"` and stops the game
- If all attempts are used up, print `"Game over! The number was 7."`

### Example output
```
Guess the number: 3
Wrong, try again.
Guess the number: 7
You got it!
```
